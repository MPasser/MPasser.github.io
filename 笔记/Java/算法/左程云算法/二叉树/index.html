二叉树结构

二叉树的递归序

对于一颗二叉树，按下列代码，访问到树节点的顺序。

对于一颗1-7的完全二叉树，其递归序即为 [1,2,4,4,4,2,5,5,5,2,1,3,6,6,6,3,7,7,7,3,1]

先序遍历（头-左-右）：在第一次访问当前节点时打印
中序遍历（左-头-右）：在第二次访问当前节点时打印
后序遍历（左-右-头）：在第三次访问当前节点时打印

非递归遍历二叉树

前序遍历

  1. 创建栈，压入根节点。
  2. 栈中弹出树节点cur，并打印。
  3. 先将cur右孩子压栈，再将cur左孩子压栈。
  4. 重复2-3步至栈空。

后序遍历

因为后序遍历和先序遍历’（头右左）是完全相反的顺序。所以直接用先序遍历’节点将节点放入收集栈中，再打印收集栈。

  1. 创建栈，压入根节点。
  2. 创建收集栈。
  3. 栈中弹出树节点cur，将节点值压入到收集栈。
  4. 先将cur左孩子压栈，再将cur右孩子压栈。
  5. 重复3-4步至栈空。
  6. 打印收集栈。

中序遍历

  1. 创建栈，压入根节点。
  2. 循环将左孩子节点压栈，即将树的左边界入栈。
  3. 栈中弹出树节点cur并打印。对cur的右孩子执行步骤2。
  4. 重复2-3步至栈空。

深度优先遍历

中序遍历即为深度优先遍历。

宽度优先遍历

  1. 创建队列，压入根节点。
  2. 队列中获取树节点cur，并打印。
  3. 先将cur左孩子添加至队列，再将cur右孩子添加至队列。
  4. 重复2-3步至栈空。

例：leetcode-662 二叉树最大宽度 maximum-width-of-binary-tree
求二叉树的最大宽度。

解1：
创建哈希表记录节点和深度的映射。宽度优先遍历树，维护当前遍历树节点的深度，和当前深度的节点数量。

解2：
宽度优先遍历。遍历过程中，维护当前深度和下一个深度的最后一个节点。为当前深度的节点数量计数，并维护最大值。

搜索二叉树

  • 节点的左子树只包含 小于 当前节点的数。
  • 节点的右子树只包含 大于 当前节点的数。
  • 所有左子树和右子树自身必须也是二叉搜索树。

搜索二叉树的中序遍历，一定为升序。

例：leetcode-98 验证二叉搜索树 validate-binary-search-tree
给定一个二叉树的根节点 root ，判断其是否是一个有效的二叉搜索树。

解：
中序遍历，判断遍历过程中节点值是否为升序。

完全二叉树

在一个 完全二叉树 中，除了最后一层外，所有层都是完全被填满的，并且最后一层中的所有节点都是尽可能靠左的。

例：leetcode-958 二叉树的完全性检验 check-completeness-of-a-binary-tree
给定一个二叉树的 root ，确定它是否是一个 完全二叉树。

解：

  • 宽度优先遍历树。
  • 若存在只有右孩子的节点，则不是完全二叉树。
  • 遇到第一个左右孩子不双全的节点后，之后的节点都必须为叶子节点，即无孩子节点。

满二叉树

除最后一层无任何子节点外，每一层上的所有结点都有两个子结点的二叉树。

用代码判断：树的节点数量 = 2^(树的深度) - 1 。

平衡二叉树

一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过1。

例：leetcode-110 平衡二叉树 balanced-binary-tree

解：
二叉树递归解题技巧：
树形DP：当前问题中，需要左右子树分别提供什么样的信息。

当前问题中，需要左右子树分别提供，子树是否是平衡二叉树，以及子树的高度。

例：leetcode-236 二叉树的最近公共祖先 lowest-common-ancestor-of-a-binary-tree
给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。
最近公共祖先的定义为：对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。

解：

  1. 设置哈希表记录节点与其父节点的映射。
  2. 回溯节点p的所有父节点，放入集合中。
  3. 回溯节点q的所有父节点，第一个在集合中的节点即为最近公共祖先。

解：
递归。
递归方法：

  • 当前节点为空，返回空。
  • 当前节点为p或q，返回p或q。
  • 否则，
  • 当前节点左右孩子都为空，返回空。
  • 当前节点左右孩子有一个不为空，返回不为空的节点。
  • 当前节点左右孩子都不为空，返回当前节点。

前驱节点和后继节点

前驱节点：中序遍历中，一个节点上一次访问的节点。
后继节点：中序遍历中，一个节点下一个要访问的节点。

例：找出后继节点
给定一个二叉树，树节点中记录了当前节点的父节点。找出当前节点的后继节点。

解
从树的结构上寻找答案。

  1. 当前节点有右子树时，其后继节点即为其右子树的最左节点。
  2. 当前节点无右子树时，判断节点是否为其父节点的左孩子，是则其父节点就是后继节点；否则一路向上找，找到第一个左孩子节点，其父节点就是后继节点。

二叉树的序列化和反序列化

将树的遍历结果存成字符串

例：
一张纸条，多次向内折叠，展开口向内的折痕称为凹，向外的折痕称为凸。给定一个整数N，按顺序输出纸条折叠N次展开后的折痕。
比如折叠一次，只有1条折痕，为 凸。
折叠两次，有3条折痕，为 凸-凸-凹。
折叠3次，有8条折痕，为 凸-凸-凹-凸-凸-凹-凹-凸。

解：
折叠的过程，就是构建满二叉树的过程，根节点为凸，多折叠一次即多加一层，所有节点的左孩子为凸，右孩子为凹。最终中序遍历输出节点即可。
