哈希

发表于 2022-04-14 分类于 算法

哈希函数

  • 输入域为无穷，输出域是有限的。
  • 相同的输入，会得到相同的输出。即哈希函数没有随机的成分。
  • 由于输入输出域的关系，必然存在输出不同，输出相同的情况，即哈希碰撞。
  • 离散性。哈希函数输出的值在输出域上整体分布均匀。

哈希表和有序表

HashSet、HashMap

  • 哈希表的增删改查都是O(1)的，但此常数略大。
  • 哈希表的key，基础类型按值传递，引用类型按引用传递。

有序表

TreeSet、TreeMap

  • 有序表的增删改查都是O(logN)的。
  • 有序表的key，基础类型按值传递，引用类型按引用传递，且需自定义比较器。

例：
设计一种数据结构，实现下列三个功能。

  • insert(key) ：将key加入到该结构中，做到不重复加入。
  • delete(key) ：将key从结构中删除。
  • getRandom() ：等概率随机返回结构中的任何一个key。

要求三种方法的时间复杂度都为O(1)。

解：
数据结构内维护两个哈希表。Map<Element, Index> 和 Map<Index, Element> 。
增加元素时直接放入Map，Index值根据size获取，递增设置。
删除元素时，将最大Index的Element替换到即将删除的元素，最大Index也可以用size找。
获取随机key可以生成[0,size-1]内的随机整数，作为Index在Map<Index, Element>中获取。
