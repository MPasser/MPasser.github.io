Integer的实例化

发表于 2021-06-30 更新于 2024-01-22

源码位置

  • java.lang.Integer
  • java version “1.8.0_191”

被弃用的构造器

public Integer(int value) 这个构造器，被标明在JDK9及之后被弃用，用户被建议使用Integer valueOf(int i)的方式实例化Integer变量。

valueOf(int i)方法

建议使用此方法的原因，在注释中有给出：

as this method is likely to yield significantly better space and time performance by caching frequently requested values.

因为通过利用缓存的值，此方法可能会显著优化空间和时间性能。

IntegerCache

IntegerCache是Integer的一个静态私有类，主要是存储了一个Integer数组，默认存储值在[-128, 127]的Integer实例。结合上一节，使用Integer valueOf(int i) 方法的时候，i如果在范围内，直接拿相应的对象，所以相同的值就是同一个对象了。不在范围内再使用new实例化。

其中存储范围的最大值，可以手动修改-XX:AutoBoxCacheMax=<size>配置项进行设置。
