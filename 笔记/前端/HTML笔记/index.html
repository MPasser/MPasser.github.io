HTML笔记

发表于 2020-07-05 分类于 Web

简介

  • HTML是什么
    • HTML：Hyper Text Markup Language ，超文本标签语言
    • 由各种标签组成，用来只做网页，告诉浏览器该如何显示页面
  • 版本
    • W3C：World Wide Web Consortium 万维网联盟，制定Web技术相关标准和规范的组织，HTML就是W3C制定的标准
    • 两个版本：HTML4.01、HTML5
    • 官网：w3school
  • 后缀名
    • HTML文件以 .html 或 .htm 为后缀

HTML文档结构

基本结构

  • HTML标签是由尖括号包围的关键词，如<html>，通常都是成对出现，如<html></html>
  • 以标签为根标签，包含<head>头部和<body>主体部分
  • 头部提供关于网页的相关信息，如标题、字符编码、关键字等摘要信息
  • 主体部分提供网页的具体内容，真正显示在页面中
  • 可以在网页文件的第一行使用<!DOCTYPE html>声明文档的类型，表示使用HTML5版本规范
  • 合理的进行缩进；标签名不区分大小写，但一般都习惯全小写

标签

组成

一个完整的HTML标签的组成：<标签名 属性名="属性值">内容</标签名>

属性值要用引号引起来，一般使用双引号，多个属性用空格分开

分类

根据标签是否关闭，可以分为：关闭型和非关闭型

  • 关闭型：html head body h1
  • 非关闭型： br hr meta

根据标签是否独占一行，可以分为块级元素和行级元素

  • 块级元素 div h1
  • 行级元素 span

注释

实体字符

也称为特殊字符，用于显示一些特殊符号，如：<、>、&、空格等。

常用实体字符：

实体字符 显示符号
&lt; <
&gt; >
&amp; &
&nbsp; 空格（对于连续的空白字符，在浏览器中显示时只会显示一个空格）
&copy; © 版权符号
&reg; ® 注册符号

实体字符名大小写敏感

常用标签

基本标签

标签 含义 说明
br 换行标签 非关闭型
p 段落标签 关闭型，块级元素，上下游明显的间隔
h1、h2…h6 标题标签 按照h1到h6逐渐变小，块级元素，加粗显示
div 分区标签 常作为容器来使用，一般用在页面布局中，块级元素
span 范围标签 默认没有任何效果，一般用来设置行内的特殊样式
ol、li 有序列表 有顺序的项目列表
ul、li 无序列表 无顺序的项目列表
hr 水平线标签 非关闭型，块级元素
img 图像标签 非关闭型，行级元素

hr标签的常用属性：

  • color颜色
    • 两种写法：
      • 颜色名称，如red、green、blue…
      • 十六进制的RBG，用法：#RRBBGG，每个颜色的范围是0-255，转换为十六进制是00-FF
        • 如#FF0000 红色、#00FF00 绿色、#FFFFFF白色等
        • 简写的要求：三种颜色各自颜色的两位值必须都相同。如：#f00、#ccc
  • size 粗细
  • width 宽度
    • 两种写法：
      • 像素，绝对值（固定值）
      • 百分比，相对值，相对于父容器的宽度百分比
  • align 对齐
    • 取值：center（默认值）、left、right

img标签的常用属性：

  • src 源，source，指定图片的路径，必须要有
    • 指定的图片默认以html文件所在文件夹为根
  • alt 当图片无法显示时的提示信息
  • title 当鼠标停留在图片上时显示的提示信息
  • width/height 设置图片的宽和高
    • 默认按图片原尺寸显示
    • 如果只设置其中一个，则另一个按比例缩放
    • 如果同时设置宽和高，可能会导致图片变形

开发者工具

用来帮助开发人员查看和调试页面，浏览器基本上都提供了类似的工具。在页面中右击 -> 检查/查看/审查元素 -> 按F12

常用工具：

  • Console：控制台，显示各种警告和错误信息
  • Elements：从浏览器的角度查看页面，浏览器渲染页面时的HTML、CSS、DOM等
  • Network：查看网络请求信息，浏览器向服务器请求了哪些资源、资源的大小以及加载资源花费的时间

超链接

使用<a>标签来创建超链接

表格

表格是一个规则的行列结构，每个表格由若干行组成，每行由若干单元格组成

table、row、column

基本结构

table标签

用来定义表格

常用属性：

  • border 边框，默认为0
  • width/height 宽度/高度
  • align 水平对齐，取值：left、center、right
  • bordercolor 边框颜色
  • bgcolor 背景颜色
  • background 背景图片
  • cellspacing 间距，单元格与单元格之间的距离
  • cellpadding 边距，单元格内容与边界之间的距离

tr标签

用来定义行，table row

常用属性：

  • align 水平对齐，取值：left、center、right
  • valign 垂直对齐，取值：top、middle、bottom
  • bgcolor 背景颜色
  • background 背景图片

td标签

用来定义单元格，table data

常用属性：align、valign、bgcolor、background

表格必须由行组成，行必须由单元格组成，数据必须放到单元格中

合并单元格

也称为表格的跨行跨列

两个属性：

  • rowspan，设置单元格所跨的行数，如rowspan=2，表示跨越2行
  • colspan，是指单元格所跨的列数，如colspan=4，表示跨越4列
  1. 在跨越的单元格中设置rowspan/colspan属性
  2. 将被跨越的单元格删除

必须要保证每行的实际列数是相同的，否则可能会出现错乱

高级标签

  • caption标签，表格的标题
  • thread标签，表格的头部 table head
  • th标签，表格的头部标题 table head title，效果：居中、加粗
  • tbody标签，表格的主题 table body（浏览器会自动为表格添加tbody）
  • tfoot标签，表格的脚部 table foot

一般用在thread中，设置头部的标题，替代td标签，与td的区别在于：加粗和居中对齐

表单

表单是一个包含若干元素的区域，用于获取不同类型的用户信息

表单元素允许用户在表单中输入信息的元素，如文本框、密码框、单选按钮、复选框、下拉列表、按钮等

基本结构

表单语法

form标签

用来定义表单，可以包含多个表单元素

常用属性：

  • action 提交数据给谁处理，默认为当前页面
  • method 提交数据的方式，取值：get（默认值）、post
  • get 和 post 的区别：
  • get：以查询字符串形式提交、地址栏能看到、长度有限制、不安全
  • post：以表单数组形式提交、地址栏中看不到、长度无限制、安全（较常用）
  • enctype 数据传输的方式，默认为application/x-www-form-uriencoded，上传文件时要改为multipart/form-data

普通表单元素

大多数表单元素都是使用<input>标签定义，通过type属性设置表单元素的类型

表单元素类型 含义 说明
text 单行文本框 省略不写时默认就是text
password 密码框 输入时不会明文显示，安全
radio 单选按钮 只能选择其中的一个
checkbox 复选框 可以同时选择多个
submit 提交按钮 提交表单数据
reset 重置按钮 重置表单元素的初始值
image 图像按钮 可以使用图片作为按钮，也具有提交的功能
button 普通按钮 默认无功能
file 文件选择器 选择要上传的文件
hidden 隐藏域 在页面上不显示，但会提交，可以用来存储数据

单行文本框

常用属性：

  • name 名称，很重要，如果没有name，则该表单元素的数据不会被提交
  • value 初始值
  • placeholder h5特性，文本框提示
  • size 宽度
  • maxlength 最大字符数，默认没有限制
  • readonly 只读，readonly=”readonly” ，可简写为readonly，即只写属性名
  • disabled 禁用
    • readonly和disabled区别：readonly的数据会提交，而disabled的数据不会提交

表单元素被提交的两个条件：1.有name属性 2.非disabled

单选按钮

常用属性：

  • name 名称，多个radio的name属性值必须相同，才能实现单选（互斥）
  • value 值
  • checked 是否选中，两种状态：选中、未选中

复选框

常用属性和radio基本一致

文件选择器

常用属性：

  • name 名称
  • accept 设置可选的文件类型，用来限制上传的文件类型

使用MIME格式字符串对资源类型进行限制

常用MIME类型：

  • 纯文本：text/html、text/css、text/javascript、text/xml、text/plain
  • 图像：image/jpeg、image/png、image/gif

可以通过类似 image/ 来匹配所有图像文件*

多媒体标签

audio标签

在页面中插入音频、不同浏览器对音频格式的支持不一样

audio属性常用标签：

  • src 音频文件的来源
  • autoplay 是否自动播放，默认不自动播放
  • controls 是否显示控制面板，默认不显示
  • loop 是否循环播放

video标签

在页面中插入视频，不同浏览器对视频格式的支持不一样

用法与audio标签基本相同，除此之外其他的属性：

  • width/height 视频播放器的尺寸
  • poster 在视频加载前显示的图片

查看 某一个标签在不同浏览器上的支持情况，可以访问网站caniuse 进行查询
