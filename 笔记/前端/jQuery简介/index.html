jQuery简介

jQuery简介

jQuery是目前最流行的JavaScript库，它的理念是：write less, do more，它有如下优势：

  • 轻量级
  • 强大的选择器
  • 出色的DOM操作封装
  • 可靠的事件处理机制
  • 出色的浏览器兼容性
  • 使用隐式迭代简化编程
  • 丰富的插件支持

jQuery功能

  • 访问和操作DOM元素
  • 控制页面样式
  • 对页面事件进行处理
  • 扩展新的jQuery插件
  • 与Ajax技术完美结合

jQuery版本

名称 说明
jquery-1.版本号.js（开发版） 完整无压缩版本，主要用于测试、学习和开发
jquery-1.min.版本号.js（发布版） 经过工具压缩或经过服务器开启Gzip压缩，主要应用于发布的产品和项目

jQuery基本语法

语法：

  • $() 工厂函数：将DOM对象转化为jQuery对象，其中$实际上是“jQuery”的别名。当$出现冲突时，可使用 var jq = jQuery.noConflict(); 更换别名。
  • selector 选择器：获取需要操作的DOM元素.
  • action() 方法：jQuery中提供的方法，其中包括绑定事件处理的方法。
  • $(document).ready(callback)表示在页面加载完成后执行callback，可以简化为$(callback)。

$(document).ready(callback)与window.onload函数的区别：

  • 调函数使用方式不同：前者是作为参数，后者是直接赋值。
  • 执行回调函数时机不同：前者在所有DOM文档结构绘制完毕后即执行，后者是在网页中所有内容加载完毕（包括图片、flash、视频等）之后才执行。

jQuery选择器

简介

选择器是jQuery的根基，在jQuery中，对事件处理、遍历DOM和Ajax操作都依赖于选择器.

jQuery选择器的优点：

  • 简洁的写法
  • 完善的事件处理机制

选择器的分类：基本选择器和过滤选择器

  • 基本选择器是jQuery中最常用的选择器，也是最简单的选择器。
  • 过滤选择器主要是通过特定的过滤规则来筛选出所需的DOM元素，过滤选择器都以冒号开头

按照不同的过滤规则，过滤选择器可以分为六种：

  • 基本过滤
  • 内容过滤
  • 可见性过滤
  • 属性过滤
  • 子元素过滤和表单对象属性过滤

基本选择器

基本选择器

选择器 描述 返回
#id 根据指定的id匹配一个元素 单个元素组成的集合
.class 根据给定的类名匹配元素 集合元素
element 根据给定的元素名匹配元素 集合元素
* 匹配所有元素 集合元素
selector1, selector2, … , selectorN 将每一个选择器匹配到的元素合并后一起返回 集合元素

层次选择器

如果想通过DOM元素之间的层次关系来获取特定元素，例如后代元素、子元素、相邻元素、兄弟元素等，则需要使用层次选择器。

选择器 描述 返回
$(“ancestor descendant”) 选取ancestor的所有descendant（后代）元素 集合元素
$(“parent > child”) 选取parent元素下的child（子）元素 集合元素
$(“prev+next”) 选取紧接在prev元素后的下一个next元素 集合元素
$(“prev ~ siblings”) 选取prev元素后的所有siblings元素 集合元素

子元素仅包括下一级元素，后代元素则包括子元素和子元素的子元素。

兄弟元素（sibling）与元素的排列顺序无关，只需要在同一级即可。

过滤选择器

基本过滤选择器

选择器 描述 返回
:first 选取第一个元素 单个元素组成的集合
:last 选取最后一个元素 单个元素组成的集合
:not(selector) 去除所有与给定选择器匹配的 元素 集合元素
:even 选取索引是偶数的所有元素，索引从0开始 集合元素
:odd 选取索引是奇数的所有元素，索引从0开始 集合元素
:eq(index) 选取索引等于index的元素，索引从0开始 集合元素
:gt(index) 选取索引大于index的元素，索引从0开始 集合元素
:lt(index) 选取索引小于index的元素，索引从0开始 集合元素
:header 选取所有的标题元素（h1~h6） 集合元素
:animated 选取当前正在执行动画的所有元素 集合元素

内容过滤选择器

选择器 描述
:contains(text) 选取含有文本内容为text的元素
:empty 选取不包含子元素或者文本的空元素
:has(selector) 选取含有选择器所匹配元素的元素
:parent 选取含有子元素或者文本的元素

属性过滤选择器

选择器 描述
[attribute] 选取含有此attribute属性的元素
[attribute = value] 选取属性的值为value的元素
[attribute != value] 选取属性的值不等于value的元素以及没有attribute属性的元素
[attribute ^= value] 选取指定属性的值以value开始的元素，相当于模糊查询
[attribute $= value] 选取指定属性的值以value结束的元素
[attribute *= value] 选取指定属性的值包含value的元素
[selector1][selector2]…[selectorN] 用属性选择器合并成一个复合属性选择器，满足多个条件，每选择一次，范围缩小一次

子元素过滤选择器

选择器 描述
:nth-child(index/even/odd/equation) 选取每个父元素下的第index个子元素或者奇偶元素（index从1算起）
:first-child 选取每个父元素的第一个元素
:last-child 选取每个父元素的最后一个元素
:only-child 如果某个元素是它父元素中唯一的子元素，那么将被匹配

可见性过滤选择器

  • 可见性过滤选择器是根据元素的可见性和不可见状态来选择相应的元素
  • 可见选择器：hidden不仅包括样式属性display为none的元素，也包含文本隐藏域（<input type="hidden">）之类的元素 ，但不包含（<span style="visibility:hidden"></span>）之类的元素
选择器 描述
:hidden 选取所有不可见的元素
:visible 选取所有可见的元素

表单选择器

选择器 描述
:input 选取所有的<input>、<textarea>、<select>和<button>元素
:text 选取所有的单行文本框
:password 选取所有的密码框元素
:radio 选取所有的单选框
:checkbox 选取所有的多选框
:submit 选取所有的提交按钮
:image 选取所有的图像按钮
:reset 选取所有的重置按钮
:button 选取所有的按钮
:file 选取所有的上传域

表单属性过滤选择器

此选择器主要对所选择的表单元素进行过滤

选择器 描述
:enabled 选取所有可用的元素

jQuery中的事件

简介

jQuery事件是对JavaScript事件的封装，常用事件分类如下：

  • 基础事件
    • windows事件
    • 鼠标事件
    • 键盘事件
    • 表单事件
  • 复合事件是多个事件的组合
    • 鼠标光标悬停
    • 鼠标连续点击

基础事件

鼠标事件

鼠标事件是当用户在文档上移动或单击鼠标时产生的事件，常用鼠标事件有：

方法 执行时机
click() 单机鼠标时
mouseover() 鼠标移过时
mouseout() 鼠标移出时

键盘事件

用户每次按下或者释放键盘上的按键时都会产生事件，常用键盘事件有：

方法 执行时机
keydown() 按下键盘时
keyup() 释放键盘时
keypress() 产生可打印的字符时

执行顺序 ：keydown > keypress > keyup

表单事件

当元素获得焦点时，会触发focus事件，时区焦点时，会触发blur事件。

方法 执行时机
focus() 获得焦点
blur() 失去焦点

jQuery中的DOM操作

简介

DOM操作分为三类：

  • DOM Core：任何一种支持DOM的编程语言都可以使用它，如getElementById() 。
  • HTML-DOM：用于处理HTML文档，如document.forms 。
  • CSS-DOM：用于操作CSS，如element.style.color=”green” 。

jQuery对JavaScript中的DOM操作进行了封装，使用起来也更简便，jQuery中的DOM操作可分为：

  • 样式操作
  • 内容及Value属性值操作
  • 节点（元素）操作
  • 节点属性操作
  • 节点遍历
  • CSS-DOM操作

节点操作

jQuery中节点操作主要分为：

  • 查找节点：通过jQuery选择器完成 。
  • 创建节点：使用jQuery工厂函数 $(tagName)，可以直接创建时添加属性和内容
  • 插入节点
方法 描述
$(‘element’).append(node) 向所有指定元素内部结尾处追加内容
$(node).appendTo(element) 将内容向指定元素内部结尾处追加
$(‘element’).prepend(node) 向所有指定元素内部开始处追加内容
$(node).prependTo(element) 将内容向指定元素内部开始处追加
$(‘element’).after(node) 向所有指定元素结尾处追加内容
$(node).insertAfter(element) 将内容向指定元素结尾处追加
$(‘element’).before(node) 向所有指定元素开始处追加内容
$(node).insertBefore(element) 将内容向指定元素开始处追加
  • 删除节点：element.remove()方法表示从DOM中删除所有匹配的元素，empty()方法表示清空元素中所有后代节点（不包含属性节点）。
  • 复制节点：clone() 用于克隆匹配的DOM元素，默认不复制事件属性，传入参数true可复制事件属性。
  • 替换节点：old.replaceWith(new) 将所有匹配的元素都替换为指定的HTML或DOM元素，replaceAll()即颠倒的replaceWith()方法，替换不会保留原先元素的事件。

节点属性操作

方法 作用
.css(“color”,”red”) 查询或改变元素样式，若要同时添加多个，则使用类似JSON的格式传入参数
.addClass(“navbar”) 为元素添加类属性，若要同时添加多个，类名间用逗号隔开
.removeClass(“navbar”) 为元素移除类属性
.val() 获取元素的value属性值，若传入参数则表明修改value属性为参数值
.attr(‘type’) 返回指定属性的值，若传入第二个参数则表明修改属性值为参数值
.removeAttr() 删除指定元素的指定属性

节点内容操作

语法 功能
.html() 用于获取指定元素的HTML内容或文本内容，若传入参数则表明修改内容为参数内容
.text() 用于获取指定元素的文本内容，若传入参数则表明修改内容为参数内容

节点遍历操作

  • 遍历子元素

children()方法可以用来获取元素的所有子元素，只考虑子元素而不考虑任何后代元素。

  • 遍历前辈元素

parent()方法获取元素的父级元素

parents()方法获取元素的祖先元素，包括父元素，父元素的父元素，…

  • 兄弟元素
    • next() 下一个兄弟元素
    • nextAll() 所有之后的兄弟元素
    • prev() 上一个兄弟元素
    • prevAll() 所有之前的兄弟元素
    • siblings() 获取所有兄弟元素
  • 使用工具函数

jQuery动画

jQuery内置动画效果

  • 显示和隐藏
    • 显示：show([speed],[callback]) 使元素集合的元素显示
      • speed：效果的持续时间，可取slow、normal、fast ，或数字表示毫秒数若省略则不产生动画
      • callback：动画结束时的回调方法
    • 隐藏：hide([speed],[callback]) 使元素集合的元素隐藏
  • 淡入和淡出
    • 淡入：fadeIn([speed],[callback]) 使元素集合的元素淡入显示
    • 淡出：fadeOut([speed],[callback]) 使元素集合的元素淡出显示
  • 滑上和滑下
    • 滑上：slideUp([speed],[callback]) 使元素集合的元素像上开窗帘一样隐藏
    • 滑下：slideDown([speed],[callback]) 使元素集合的元素像下拉窗帘一样显示

自定义动画效果

调用animate()方法可以创建自定义动画效果，它的调用格式为：

  • params ：CSS属性名与值，动画要达到的效果值，必选
  • speed ：效果的持续时间，单位为毫秒
  • callback：动画结束时的回调方法
