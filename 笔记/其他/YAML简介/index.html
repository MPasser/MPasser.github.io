YAML简介

发表于 2022-08-02 分类于 YAML

简介

YAML全称是 YAML Ain’t Markup Language 。YAML是一种直观能够被电脑识别的数据序列化格式，且容易被人类阅读，容易和脚本语言交互。支持导入YAML的编程语言有：C/C++、Ruby、Python、Java、Perl、C#和PHP等。

YAML文件是以数据为核心的，比传统的XML方式更加简洁。

YAML文件的扩展名可以使用 .yml 或 .yaml

基本语法

  • 大小写敏感
  • 数据值前必须有空格作为分隔符
  • 使用缩进表示层级关系
  • 缩进使用空格且不能使用Tab代替，缩进空格数目不做要求，相同层级元素左侧对齐即可
  • 使用#表示注释，当前行此字符后的内容会被解析器忽略

多文件

可以将多个YAML文件合并为一个文件。在使用 profile 配置时就无需创建多个文件了。

不同文件之间，使用 ---进行分隔。
